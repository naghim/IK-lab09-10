# Klaszterezés

A klaszterezés a nem felügyelt gépi tanulás csoportjába sorolható, az algoritmus során egy adathalmaz pontjait, objektumait hasonlóság alapján csoportosítjuk, úgy hogy az egy csoportban lévő objektumok minél inkább egymáshoz hasonlóak, míg a más csoportokban lévők pedig különbözőek legyenek.

A labor célja, hogy összehasonlítsuk több klaszterezési módszer viselkedését különböző adathalmazokon, és értelmezzük az eredményeket vizualizációk és metrikák alapján.
